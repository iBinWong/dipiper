# 获取股票列表

  • 接口用途
    获取沪深 A 股列表,由于返回的是所有股票，整理数据的时间较长，建议调用后自行存储，方便其他接口需要股票代码作为参数时使用。

  • 接口调用说明

    dip.stock.symbols.getStockList().then((data) => {
    //数据存储、处理逻辑，请自行实现
});

  • 参数说明
    没有输入参数

  • 返回说明 正常时的返回 JSON 数据包示例：

    [
    {
    "symbol": "sh600000",
    "code": "600000",
    "name": "浦发银行"
    },
    {
        "symbol": "sh600004",
        "code": "600004",
        "name": "白云机场"
    },
    ...
]

    返回参数说明

    参数说明
    symbol带市场标识的股票代码,其他数据接口中可能会用到
    code不带市场标识的股票代码
    name股票名称

# 股票发行信息

  • 接口用途
    获取股票的上市发行信息

  • 调用说明

    dip.stock.symbols.getStockIssue("000725").then((data) => {
    //数据存储、处理逻辑，请自行实现
});

  • 参数说明

    参数说明
    code股票代码（不带市场标识）

  • 返回说明 正常时的返回 JSON 数据包示例：

    {
    "stock_issue": [
        {
            "market": "深圳证券交易所",
            "consignee": "南方证券股份有限公司",
            "underwriting": "余额包销",
            "sponsor": "南方证券有限公司",
            "issue_price": "16.80",
            "issue_mode": "网下询价配售、网上定价发行",
            "issue_p/e": "--",
            "pre_capital": "48,955.40",
            "capital": "54,955.40",
            "issue_volume": "6,000.00",
            "expected_fundraising": "100,800.00",
            "fundraising": "100,800.00",
            "issue_cost": "3,310.00",
            "net_amount_raised": "97,490.00",
            "underwriting_fee": "--",
            "announcement_date": "2000-12-11",
            "launch_date": "2001-01-12"
        }
    ]
}

    返回参数说明

    参数说明
    market上市地
    consignee主承销商
    underwriting承销方式
    sponsor上市推荐人
    issue_price每股发行价(元)
    issue_mode发行方式
    issue_p/e发行市盈率（按发行后总股本）
    pre_capital首发前总股本（万股）
    capital首发后总股本（万股）
    issue_volume实际发行量（万股）
    expected_fundraising预计募集资金（万元）
    fundraising最实际募集资金合计（万元）
    issue_cost发行费用总额（万元）
    net_amount_raised募集资金净额（万元）
    underwriting_fee承销费用（万元）
    announcement_date招股公告日
    launch_date上市日期
